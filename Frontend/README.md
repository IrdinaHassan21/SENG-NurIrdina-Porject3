# Cat Collector

**Cat Collector** is a small 2D game made using HTML, CSS, and JavaScript with the Canvas API. The player moves around the canvas to collect good cats while avoiding bad cats.  

This project was created as part of an assignment to demonstrate skills with the Canvas API, data structures, functions, and basic algorithms.

---

## How to Play

1. Press the **Start Game** button to begin.  
2. Use the **arrow keys** or **WASD keys** to move the character.  
3. **Good cats** give +1 point.  
4. **Bad cats** give -1 point. 
5. **Chonky cats** give +2 point. 
6. The game lasts **60 seconds**, after which the final score is displayed.
7. The player speed will increase every 10 seconds.  
8. Press **Restart Game** anytime to play again.  

---

## Features

- Player movement with keyboard  
- Randomly spawning good and bad cats every second  
- Score and high score tracking  
- Timer countdown  
- Cats disappear after collection  
- Game Over screen  
- Sidebar with instructions  
- Garden background and pixel-style graphics  

---

## Technologies Used

- **HTML** – structure  
- **CSS** – styling  
- **JavaScript** – game logic  
- **Canvas API** – drawing and animation  
- **Arrays** – storing cats  
- **Functions** – organizing code
